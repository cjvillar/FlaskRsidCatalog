# This is a basic flask app set up project.
### To get started:
1. Clone Repo.
2. Create a VENV, python3 -m venv venv
3. . venv/bin/activate
4. pip install Flask
5. pip install -r requirments
6. export FLASK_APP=hello.py
7. export 'PYTHONPATH="${PYTHONPATH}:/Users/chris/flask_project'
8. flask Run
